# friday

#### 介绍
基于Spring Boot2.X的后台权限管理系统，集成了Spring Security，实现了简单RBAC模型，可作为开发项目的脚手架，作为基础项目。

#### 软件技术
1. Spring Boot 2.1.4
2. Spring Security 5.1.5
3. MybatisPlus 3.0.5
4. Logback 1.2.3
5. Druid 1.1.10
6. Swagger 2.9.2
7. Lombok 1.18.6
8. X-admin 2.2
9. Thymeleaf 3.0.11
10. Layui 2.5.3
11. MySQL 5.6

#### 内置功能
1. 用户管理：用户查询、添加用户、修改用户、用户角色设置、删除用户；
2. 角色管理：角色查询、添加角色、修改角色、角色菜单权限配置、删除角色；
3. 菜单管理：菜单查询、添加菜单、修改菜单、删除菜单；
4. 登录、登出：基于Spring Security的认证和授权；
5. Druid数据源监控功能；
6. Swagger接口文档功能；
7. 修改密码；
